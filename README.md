## Projeto de Faculdade

### Descrição

O projeto é uma API de gestão de oficina mecânica, que visa melhorar o rendimento e eficiência dos processos realizados em uma oficina mecância. O sistema permite que possam gerenciar funcionários, gerenciar os clientes, gerenciar os veículos dos clientes e gerenciar o estoque. Realizando o processos de orçamento e realização de serviços somente se o orçmaneto for aprovado.
